# 🌙 Dream Diary

一个用于记录梦境与日常想法的轻量级日记应用，支持创建、查看和管理个人梦境记录，界面简洁，交互流畅。

---

## 📌 项目简介

Dream Diary 是一个面向个人使用的日记记录系统，用户可以记录自己的梦境、灵感或日常想法。

项目注重：

- ✨ 简洁美观的界面设计
- 🧠 沉浸式记录体验
- 📱 轻量化与易用性

适合作为：

- 前端练习项目
- Web 应用课程设计
- 个人作品集展示项目

---

## ✨ 功能特性

- 📝 新建梦境记录
- 📖 查看历史记录
- 🗑️ 删除记录
- ⏰ 时间自动记录
- 🎨 简洁 UI + 良好交互体验

---

## 🚀 快速开始

### 1️⃣ 克隆项目

```bash
git clone https://github.com/你的用户名/dream-diary.git
cd dream-diary
```
2️⃣ 运行项目

直接打开：
```
index.html
```
或使用本地服务器：
```
live-server
```
🧠 核心实现思路
数据存储

使用浏览器 localStorage 保存日记内容

数据结构示例：
```

[
  {
    "id": 1,
    "content": "梦见自己在飞",
    "time": "2026-03-20 22:00"
  }
]
```
页面逻辑

输入内容 → 保存

渲染列表 → 展示所有记录

删除按钮 → 更新数据并重新渲染
